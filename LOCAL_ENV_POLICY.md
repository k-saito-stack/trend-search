# ローカル .env 廃止ポリシー

このリポジトリは、APIキーを GitHub Secrets で管理します。
そのため、ローカルの `.env` / `.env.example` は使いません。

## Macで `git pull` した後にやること

1. まず1回だけ実行:
   - `npm run doctor:local`
2. 以降、`.env` が残っている状態で `npm test` などを実行すると、
   チェックスクリプトが警告して止まります。

## 補足

- `git pull` だけでは、Git管理外ファイル（`.env`）は自動削除されません。
- そのため、このチェックを入れて「残っていたら必ず気づく」ようにしています。
